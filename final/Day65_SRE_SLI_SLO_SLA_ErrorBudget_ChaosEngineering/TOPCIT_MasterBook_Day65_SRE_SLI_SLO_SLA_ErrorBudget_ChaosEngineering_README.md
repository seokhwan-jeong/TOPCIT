# 1. SRE란?

SRE(Site Reliability Engineering)는 소프트웨어를 이용해 안정적인
서비스를 운영하는 방법론이다.

목표 - 높은 가용성 - 자동화 - 운영 효율화 - 장애 최소화

------------------------------------------------------------------------

# 2. SLI / SLO / SLA

  항목   의미          예시
  ------ ------------- -----------------------
  SLI    서비스 지표   응답시간, 성공률
  SLO    목표 수준     가용성 99.9%
  SLA    고객 계약     월 99.9% 미만 시 보상

SLI → SLO → SLA 순으로 연결된다.

------------------------------------------------------------------------

# 3. Error Budget

Error Budget = 100% - SLO

예) - SLO = 99.9% - Error Budget = 0.1%

활용 - 신규 기능 출시 여부 결정 - 운영 안정성 평가

------------------------------------------------------------------------

# 4. Incident Response

절차

``` text
탐지
 ↓
분류
 ↓
대응
 ↓
복구
 ↓
RCA
 ↓
재발 방지
```

우선순위 - P1 : 서비스 중단 - P2 : 핵심 기능 장애 - P3 : 일반 장애

------------------------------------------------------------------------

# 5. RCA(Root Cause Analysis)

대표 기법

-   5 Whys
-   Fishbone Diagram
-   Timeline Analysis

목적 - 근본 원인 제거 - 재발 방지

------------------------------------------------------------------------

# 6. Toil Reduction

Toil은 반복적이고 자동화 가능한 운영 작업이다.

예시 - 수동 배포 - 반복 로그 확인 - 계정 생성

자동화 도구 - Ansible - Terraform - ArgoCD

------------------------------------------------------------------------

# 7. Chaos Engineering

운영 중 의도적으로 장애를 발생시켜 시스템 복원력을 검증한다.

대표 도구 - Chaos Monkey - LitmusChaos - Gremlin

# TOPCIT 실전 문제

## 문제1 (중)

가용성 목표가 99.95%라면 무엇에 해당하는가?

① SLI ② SLO ③ SLA ④ KPI

**정답:** ②

**해설:** SLO는 서비스 목표 수준을 의미한다.

------------------------------------------------------------------------

## 문제2 (상)

반복적인 운영 업무를 자동화하는 SRE 활동은?

① Toil Reduction ② RAID ③ NAT ④ ETL

**정답:** ①

**해설:** Toil은 반복 작업이며 자동화 대상이다.

------------------------------------------------------------------------

## 문제3 (상)

장애의 근본 원인을 분석하는 활동은?

① RCA ② DHCP ③ VPN ④ CDN

**정답:** ①

------------------------------------------------------------------------

## 문제4 (시나리오)

운영 중 일부 서버를 의도적으로 종료하여 시스템 복원력을 검증하려 한다.
적합한 방법은?

① Chaos Engineering ② Backup ③ RAID1 ④ FTP

**정답:** ①

**해설:** Chaos Engineering은 실제 장애 상황을 모의해 복원력을 검증한다.

------------------------------------------------------------------------

## 문제5 (계산형)

서비스 SLO가 99.99%라면 Error Budget은?

① 0.1% ② 0.01% ③ 1% ④ 10%

**정답:** ②

**해설:** Error Budget = 100% - 99.99% = 0.01%.
