# 1. 네트워크란?

네트워크(Network)는 여러 장치가 데이터를 주고받을 수 있도록 연결한
환경이다.

대표 구성요소

-   Client
-   Server
-   Switch
-   Router
-   Firewall

``` text
PC
 │
L2 Switch
 │
L3 Switch / Router
 │
Internet
 │
Server
```

------------------------------------------------------------------------

# 2. OSI 7계층

  계층   이름           대표 프로토콜
  ------ -------------- ---------------
  L7     Application    HTTP, DNS
  L6     Presentation   TLS, SSL
  L5     Session        RPC
  L4     Transport      TCP, UDP
  L3     Network        IP, ICMP
  L2     Data Link      Ethernet, ARP
  L1     Physical       Cable, Fiber

### 암기법

**응표세전네데물** (응용-표현-세션-전송-네트워크-데이터링크-물리)

------------------------------------------------------------------------

# 3. TCP/IP 모델

  TCP/IP           OSI 대응
  ---------------- ----------
  Application      L5\~L7
  Transport        L4
  Internet         L3
  Network Access   L1\~L2

OSI는 참조 모델, TCP/IP는 실제 인터넷에서 사용하는 모델이다.

------------------------------------------------------------------------

# 4. Ethernet과 MAC Address

Ethernet은 가장 널리 사용되는 LAN 기술이다.

MAC Address 특징

-   48bit
-   제조사 고유값
-   L2에서 사용

예시

``` text
00:1A:2B:3C:4D:5E
```

------------------------------------------------------------------------

# 5. IP Address

IP Address는 네트워크상의 논리 주소이다.

IPv4 예

``` text
192.168.10.15
```

차이점

  MAC              IP
  ---------------- -----------
  물리 주소        논리 주소
  변경 거의 없음   변경 가능
  L2               L3

------------------------------------------------------------------------

# 6. MTU

MTU(Maximum Transmission Unit)는 한 번에 전송 가능한 최대 데이터
크기이다.

기본 Ethernet MTU

``` text
1500 Bytes
```

Jumbo Frame

``` text
9000 Bytes
```

### 실무 사례

MTU가 서로 다르면

-   Fragmentation
-   Packet Drop
-   성능 저하

가 발생할 수 있다.

# 예상 문제

## 문제 1

TCP는 어느 계층에서 동작하는가?

1.  L2
2.  L3
3.  L4
4.  L7

**정답:** 3

### 상세 해설

TCP는 전송 계층(L4) 프로토콜로 신뢰성 있는 통신을 제공한다. L3는 IP,
L2는 Ethernet, L7은 HTTP 등이 대표적이다.

------------------------------------------------------------------------

## 문제 2

MAC Address의 특징으로 올바른 것은?

1.  논리 주소이다.
2.  L3에서 사용한다.
3.  48bit 물리 주소이다.
4.  DNS가 할당한다.

**정답:** 3

### 상세 해설

MAC Address는 네트워크 인터페이스 카드(NIC)에 부여되는 48bit 물리
주소이며 데이터링크 계층에서 사용된다.

------------------------------------------------------------------------

## 문제 3

MTU가 서로 다른 장비 간 통신 시 발생 가능한 문제는?

1.  CPU Overload
2.  Fragmentation
3.  RAID 장애
4.  Cache Miss

**정답:** 2

### 상세 해설

패킷 크기가 MTU보다 크면 조각화(Fragmentation)가 발생하거나 Drop될 수
있으며 성능 저하의 원인이 된다.

------------------------------------------------------------------------

## 문제 4

OSI 7계층에서 IP가 속하는 계층은?

1.  L2
2.  L3
3.  L4
4.  L7

**정답:** 2

### 상세 해설

IP는 Network Layer(L3) 프로토콜이며 라우팅과 논리 주소를 담당한다.

------------------------------------------------------------------------
