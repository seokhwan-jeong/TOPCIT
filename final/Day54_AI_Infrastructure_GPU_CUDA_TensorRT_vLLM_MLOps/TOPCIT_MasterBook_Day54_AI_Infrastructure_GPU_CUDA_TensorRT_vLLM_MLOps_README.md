# 3. CUDA

CUDA는 NVIDIA GPU를 활용하기 위한 병렬 컴퓨팅 플랫폼이다.

장점 - Tensor 연산 가속 - AI 학습 속도 향상 - 병렬 처리

------------------------------------------------------------------------

# 4. TensorRT

TensorRT는 AI 추론을 최적화하는 엔진이다.

기능 - 모델 최적화 - FP16/INT8 최적화 - 추론 지연시간 감소

------------------------------------------------------------------------

# 5. vLLM

vLLM은 대규모 언어모델 추론을 위한 고성능 엔진이다.

특징 - PagedAttention - 높은 GPU 활용률 - 대량 동시 요청 처리

------------------------------------------------------------------------

# 6. AI Inference Server

대표 솔루션

-   NVIDIA Triton
-   vLLM
-   Ollama
-   TensorRT-LLM

역할 - 모델 로딩 - 요청 처리 - 배치 추론 - 모니터링

------------------------------------------------------------------------

# 7. Kubernetes + AI

``` text
Ingress
   │
AI API
   │
Inference Pod
   │
GPU Node
```

GPU Operator와 NVIDIA Device Plugin을 사용하여 GPU를 관리한다.

------------------------------------------------------------------------

# 8. MLOps

MLOps는 ML 모델의 개발, 배포, 운영을 자동화하는 방법론이다.

구성 - 데이터 관리 - 모델 학습 - 모델 배포 - 모니터링 - 재학습

대표 도구 - MLflow - Kubeflow - Argo Workflows
