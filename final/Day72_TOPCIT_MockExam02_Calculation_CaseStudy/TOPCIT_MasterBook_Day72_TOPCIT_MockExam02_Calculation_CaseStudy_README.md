# 실전 문제

## 문제 1 (계산형)

PV=200, EV=180, AC=240이다. SPI와 CPI는?

① SPI=0.9, CPI=0.75 ② SPI=1.2, CPI=0.9 ③ SPI=0.75, CPI=0.9 ④ SPI=1.0,
CPI=1.0

**정답:** ①

**해설:** SPI=EV/PV=180/200=0.9, CPI=EV/AC=180/240=0.75. 일정과 비용
모두 계획보다 좋지 않다.

------------------------------------------------------------------------

## 문제 2 (계산형)

서비스 목표가 SLO 99.95%라면 Error Budget은?

① 5% ② 0.5% ③ 0.05% ④ 0.005%

**정답:** ③

**해설:** Error Budget = 100%-99.95%=0.05%.

------------------------------------------------------------------------

## 문제 3 (사례형)

전자상거래 서비스에서 주문과 결제를 느슨하게 연결하려고 한다. 가장
적합한 기술은?

① Kafka ② FTP ③ RAID ④ NAT

**정답:** ①

**해설:** 이벤트 기반 아키텍처(EDA)에서 Kafka는 비동기 이벤트 전달에
적합하다.

------------------------------------------------------------------------

## 문제 4 (사례형)

Kubernetes Pod가 정상 실행되지만 외부에서 접근되지 않는다. 가장 먼저
확인할 것은?

① Service와 Ingress 설정 ② CPU 교체 ③ RAID 구성 ④ DB 인덱스

**정답:** ①

**해설:** Pod가 정상이라면 네트워크 노출(Service/Ingress) 구성을 우선
확인한다.

------------------------------------------------------------------------

## 문제 5 (사례형)
