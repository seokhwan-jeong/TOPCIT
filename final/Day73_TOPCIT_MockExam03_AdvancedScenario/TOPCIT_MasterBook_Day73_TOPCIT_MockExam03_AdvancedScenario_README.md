# 실전 문제

### 1.

은행 시스템을 Multi-Region으로 전환하려 한다. 가장 중요한 목적은?

① 성능 향상 ② 재해복구(DR) ③ DB 정규화 ④ RAID 구성

**정답:** ②

------------------------------------------------------------------------

### 2.

Kubernetes에서 Pod는 Running이지만 서비스 접속이 되지 않는다. 가장 먼저
확인할 것은?

① Service / Ingress ② CPU 클럭 ③ RAID ④ JVM 버전

**정답:** ①

------------------------------------------------------------------------

### 3.

CI/CD에서 컨테이너 이미지 취약점을 검사하는 가장 적절한 도구 유형은?

① SCA ② Image Scanner(예: Trivy) ③ DHCP ④ FTP

**정답:** ②

**해설:** 오픈소스 라이브러리는 SCA, 컨테이너 이미지는 Image Scanner가
적합하다.

------------------------------------------------------------------------

### 4.

LLM이 최신 사내 규정을 모른다. 가장 적절한 개선 방법은?

① GPU 추가 ② RAG 적용 ③ CPU 추가 ④ RAID 변경

**정답:** ②

------------------------------------------------------------------------

### 5.

SPI=1.1, CPI=0.8이다. 올바른 해석은?

① 일정 지연, 비용 절감 ② 일정 양호, 비용 초과 ③ 일정 지연, 비용 초과 ④
모두 양호

**정답:** ②

------------------------------------------------------------------------

### 6.

서비스 간 암호화 통신과 트래픽 제어를 제공하는 기술은?

① Service Mesh ② CDN ③ NAT ④ VLAN

**정답:** ①

------------------------------------------------------------------------

### 7.

데이터 손실을 거의 허용하지 않는 금융 DR에 적합한 복제는?

① 동기 복제 ② 비동기 복제 ③ Full Backup ④ Tape

**정답:** ①

------------------------------------------------------------------------

### 8.

프로젝트별 클라우드 비용 분석의 핵심은?

① Tagging Strategy ② RAID10 ③ DNS ④ ICMP

**정답:** ①

------------------------------------------------------------------------

### 9.

BI 화면의 수치가 잘못되었다. 원본부터 변환 과정을 추적하려면?

① Data Lineage ② Metadata ③ Snapshot ④ ETL 서버 재부팅

**정답:** ①

------------------------------------------------------------------------

### 10.

Zero Trust의 핵심 원칙은?

① 내부망은 모두 신뢰 ② Never Trust, Always Verify ③ VPN만 사용 ④ 방화벽
제거

## 시험 직전 암기

-   Multi-Region = DR
-   Service + Ingress = 외부 노출
-   Trivy = 이미지 스캔
-   RAG = 최신 문서 기반 생성
-   SPI\>1 일정 양호 / CPI\<1 비용 초과
-   Service Mesh = mTLS
-   동기 복제 = 낮은 RPO
-   Data Lineage = 데이터 흐름
-   Zero Trust = 지속 검증
