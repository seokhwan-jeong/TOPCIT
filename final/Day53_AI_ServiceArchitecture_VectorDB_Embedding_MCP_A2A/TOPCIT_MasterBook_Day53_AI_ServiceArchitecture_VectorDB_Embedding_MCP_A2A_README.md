# 2. Embedding

Embedding은 텍스트를 의미를 보존한 벡터로 변환하는 기술이다.

활용 - 의미 기반 검색 - 추천 시스템 - 유사도 계산

------------------------------------------------------------------------

# 3. Vector Database

  제품       특징
  ---------- ------------------
  pgvector   PostgreSQL 확장
  Milvus     대규모 벡터 검색
  Pinecone   관리형 서비스
  Weaviate   AI 친화적

장점 - 빠른 유사도 검색 - RAG 기반 검색 최적화

------------------------------------------------------------------------

# 4. Prompt Engineering

대표 기법 - Zero-shot - One-shot - Few-shot - Chain of Thought(CoT) -
Role Prompting

목표 - 정확도 향상 - 환각(Hallucination) 감소

------------------------------------------------------------------------

# 5. AI Agent

AI Agent는 목표를 달성하기 위해 계획, 추론, 도구 호출을 수행하는
시스템이다.

기능 - 계획 수립 - Tool 사용 - 반복 실행 - 결과 검증

------------------------------------------------------------------------

# 6. MCP(Model Context Protocol)

MCP는 LLM이 외부 도구와 데이터에 표준 방식으로 연결되도록 하는
프로토콜이다.

장점 - 도구 연동 표준화 - 재사용성 향상 - 보안성 강화

------------------------------------------------------------------------

# 7. A2A(Agent-to-Agent)

여러 AI Agent가 역할을 분담하여 협업하는 구조이다.

예시 - 일정 Agent - 메일 Agent - 검색 Agent - 보고서 Agent

------------------------------------------------------------------------

# 8. AI 보안

점검 항목 - Prompt Injection - 데이터 유출 - 권한 제어 - 개인정보 보호 -
모델 출력 검증

## 문제1 (중 \| ★★★★★)

RAG에서 문서를 의미 기반으로 검색하는 저장소는?

① RAID ② Vector Database ③ DHCP ④ Trigger

**해설:** Embedding된 벡터를 저장하고 유사도 검색을 수행한다.

**정답: ②**

------------------------------------------------------------------------

## 문제3 (상 \| ★★★★★)
