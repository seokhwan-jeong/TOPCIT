# 1. Zero Trust

원칙: **Never Trust, Always Verify**

핵심 요소 - MFA - 최소 권한 - 지속적 검증 - Micro Segmentation - Device
Trust

------------------------------------------------------------------------

# 2. EDR / XDR

  기술   역할
  ------ ---------------------------------------
  EDR    단말 탐지·대응
  XDR    단말+네트워크+메일+클라우드 통합 탐지

XDR은 EDR보다 넓은 범위를 분석한다.

------------------------------------------------------------------------

# 3. SIEM

SIEM(Security Information and Event Management)

기능 - 로그 수집 - 이벤트 상관 분석 - 실시간 경보 - 대시보드 -
컴플라이언스

대표 제품 - Splunk - IBM QRadar - Microsoft Sentinel

------------------------------------------------------------------------

# 4. SOAR

기능 - 플레이북 실행 - 자동 대응 - 티켓 생성 - 반복 업무 자동화

SIEM과 연계되어 대응 속도를 높인다.

------------------------------------------------------------------------

# 5. SOC

SOC(Security Operations Center)

업무 - 24×7 모니터링 - 사고 대응 - 위협 헌팅 - 포렌식 지원 - 보고

------------------------------------------------------------------------

# 6. MITRE ATT&CK

주요 단계 - Initial Access - Execution - Persistence - Privilege
Escalation - Defense Evasion - Credential Access - Discovery - Lateral
Movement - Exfiltration - Impact

공격자의 TTP를 체계화한 프레임워크이다.

# TOPCIT 실전 문제

## 문제1 (중 \| ★★★★★)

'Never Trust, Always Verify'를 핵심 원칙으로 하는 보안 모델은?

① VPN ② Zero Trust ③ IDS ④ RAID

**정답:** ②

**해설:** Zero Trust는 내부 사용자도 기본적으로 신뢰하지 않는다.

------------------------------------------------------------------------

## 문제2 (상 \| ★★★★★)

단말뿐 아니라 네트워크와 클라우드까지 통합 분석하는 기술은?

① EDR ② XDR ③ SAST ④ ETL

**정답:** ②

**해설:** XDR은 다양한 보안 데이터를 통합해 분석한다.

------------------------------------------------------------------------

## 문제3 (상 \| ★★★★★)

로그를 수집하고 상관 분석하는 플랫폼은?

① SOAR ② SIEM ③ DHCP ④ Kafka

**정답:** ②

------------------------------------------------------------------------

## 문제4 (시나리오 \| ★★★★★)

반복적인 악성 IP 차단을 자동화하려면?

① SOAR 플레이북 ② RAID6 ③ NAT ④ Trigger

**정답:** ①

**해설:** SOAR는 탐지 후 자동 대응을 수행한다.

------------------------------------------------------------------------

## 문제5 (시나리오 \| ★★★★★)

랜섬웨어 공격을 MITRE ATT&CK 기준으로 분석하려 한다. 가장 큰 장점은?

① DB 성능 향상 ② 공격자의 전술·기술을 체계적으로 분석 ③ 백업 속도 향상 ④
GPU 사용량 감소

**정답:** ②

**해설:** MITRE ATT&CK는 공격자의 행동(TTP)을 기반으로 탐지와 대응
전략을 수립한다.
