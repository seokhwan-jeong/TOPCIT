# 3. NVLink / NVSwitch

  기술       특징
  ---------- -------------------
  PCIe       범용 연결
  NVLink     GPU-GPU 고속 연결
  NVSwitch   다수 GPU 스위칭

NVLink는 PCIe보다 높은 대역폭을 제공한다.

------------------------------------------------------------------------

# 4. InfiniBand

InfiniBand는 초고속 저지연 네트워크이다.

특징 - RDMA 지원 - 낮은 지연시간 - HPC/AI 환경 최적

Ethernet보다 AI 학습 성능이 우수한 경우가 많다.

# TOPCIT 실전 문제

## 문제1 (중 \| ★★★★★)

GPU 간 고속 통신 기술은?

① SATA ② NVLink ③ USB ④ DHCP

**정답:** ②

**해설:** NVLink는 GPU 간 대용량 데이터를 빠르게 전송한다.

------------------------------------------------------------------------

## 문제2 (상 \| ★★★★★)

AI 학습용 저지연 네트워크는?

① Ethernet Hub ② InfiniBand ③ NAT ④ FTP

**정답:** ②

**해설:** RDMA 기반 InfiniBand는 HPC와 AI 학습에 널리 사용된다.

------------------------------------------------------------------------

## 문제3 (상 \| ★★★★★)

GPU 하나를 여러 인스턴스로 분할하는 기술은?

① RAID ② MIG ③ VLAN ④ ETL

**정답:** ②

------------------------------------------------------------------------

## 문제4 (시나리오 \| ★★★★★)

LLM 학습 중 GPU 간 통신 병목이 발생했다. 가장 적절한 개선책은?

① NVLink/NVSwitch 활용 ② HDD 증설 ③ DNS 변경 ④ Trigger 생성

**정답:** ①

**해설:** GPU 간 통신 대역폭을 높여 학습 시간을 단축할 수 있다.

------------------------------------------------------------------------

## 문제5 (시나리오 \| ★★★★★)

AI 데이터센터의 전력 비용이 급증하고 있다. 가장 먼저 확인해야 할 지표는?

① PUE ② TTL ③ MTU ④ CIDR

**정답:** ①

**해설:** PUE는 데이터센터 전력 효율을 나타내는 핵심 지표이다.
