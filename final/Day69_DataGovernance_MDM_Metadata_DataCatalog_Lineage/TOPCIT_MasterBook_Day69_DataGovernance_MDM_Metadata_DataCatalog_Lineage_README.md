# 1. Data Governance

Data Governance는 조직의 데이터를 일관성 있고 안전하게 관리하기 위한
정책과 체계이다.

목표

-   데이터 품질 향상
-   표준화
-   보안 및 개인정보 보호
-   데이터 활용 극대화

주요 역할

-   Data Owner
-   Data Steward
-   Data Custodian

------------------------------------------------------------------------

# 2. Data Quality

대표 품질 요소

  항목           설명
  -------------- -----------
  Accuracy       정확성
  Completeness   완전성
  Consistency    일관성
  Timeliness     적시성
  Validity       유효성
  Uniqueness     중복 없음

------------------------------------------------------------------------

# 3. Master Data Management(MDM)

MDM은 고객, 제품, 조직 등 핵심 데이터를 통합 관리하는 체계이다.

효과

-   데이터 중복 제거
-   Single Source of Truth 확보
-   업무 일관성 향상

대표 마스터 데이터

-   고객
-   상품
-   조직
-   계정

------------------------------------------------------------------------

# 4. Metadata Management

Metadata는 데이터를 설명하는 데이터이다.

예시

-   테이블명
-   컬럼명
-   데이터 타입
-   생성일
-   소유자

활용

-   데이터 검색
-   영향도 분석
-   표준 관리

------------------------------------------------------------------------

# 5. Data Catalog

Data Catalog는 조직의 데이터 자산을 검색하고 관리하는 시스템이다.

기능

-   데이터 검색
-   메타데이터 관리
-   권한 관리
-   데이터 분류

대표 제품

-   Microsoft Purview
-   AWS Glue Data Catalog
-   Apache Atlas

------------------------------------------------------------------------

# 6. Data Lineage

Data Lineage는 데이터가 생성·변환·활용되는 흐름을 추적하는 기능이다.

``` text
Source
   ↓
ETL
   ↓
Data Warehouse
   ↓
Dashboard
```

장점

-   영향도 분석
-   장애 분석
-   규제 대응

# TOPCIT 실전 문제

## 문제1 (중)

조직의 데이터 정책과 표준을 관리하는 체계는?

① Data Governance ② RAID ③ Scrum ④ VLAN

**정답:** ①

**해설:** Data Governance는 데이터 정책, 표준, 품질을 관리하는 체계이다.

------------------------------------------------------------------------

## 문제2 (상)

고객 정보의 중복을 제거하고 단일 기준 정보를 유지하기 위한 시스템은?

① Data Lake ② MDM ③ VPN ④ NAT

**정답:** ②

**해설:** MDM은 핵심 데이터를 통합하여 Single Source of Truth를
제공한다.

------------------------------------------------------------------------

## 문제3 (상)

데이터의 생성부터 활용까지 흐름을 추적하는 기능은?

① Metadata ② Data Lineage ③ Snapshot ④ RAID

**정답:** ②

------------------------------------------------------------------------

## 문제4 (시나리오)

BI 대시보드의 값이 잘못 표시되고 있다. 원본 데이터를 추적하기 위해 가장
필요한 기능은?

① Data Lineage ② FTP ③ SMTP ④ DNS

**정답:** ①

**해설:** Data Lineage는 데이터 이동 경로와 변환 과정을 추적하여 원인을
분석한다.

------------------------------------------------------------------------

## 문제5 (시나리오)

AI 학습 데이터에 개인정보가 포함되어 있다. 가장 먼저 수행해야 할 작업은?

① 비식별화 및 개인정보 보호 조치 ② 데이터 삭제 ③ 서버 교체 ④ CPU 증설

**정답:** ①

**해설:** AI 데이터 거버넌스에서는 개인정보 보호와 비식별화가 우선이다.
