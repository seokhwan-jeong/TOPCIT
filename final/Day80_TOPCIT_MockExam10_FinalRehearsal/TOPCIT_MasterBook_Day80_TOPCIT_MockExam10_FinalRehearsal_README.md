# 실전 문제 (20문항)

1.  TO-BE 아키텍처 수립 전 반드시 수행해야 하는 활동은? ① AS-IS 분석 ②
    Failover ③ 운영 종료 ④ 모의해킹 **정답:** ①

2.  GitOps에서 단일 진실 공급원(Source of Truth)은? ① Git Repository ②
    DB ③ NAS ④ Wiki **정답:** ①

3.  Kubernetes에서 외부 HTTPS 요청을 처리하는 리소스는? ① Ingress ②
    ConfigMap ③ Secret ④ PVC **정답:** ①

4.  컨테이너 이미지 취약점 분석 도구는? ① Trivy ② SonarQube ③ Grafana ④
    Helm **정답:** ①

5.  오픈소스 라이브러리 취약점 분석은? ① SCA ② IDS ③ IPS ④ DAST
    **정답:** ①

6.  최신 사내 규정을 AI 답변에 반영하는 최적 방법은? ① RAG 적용 ② GPU
    증설 ③ RAM 증설 ④ Scale Up **정답:** ①

7.  서비스 간 암호화 통신과 정책 제어는? ① Service Mesh(mTLS) ② NAT ③
    VLAN ④ FTP **정답:** ①

8.  Prometheus의 핵심 기능은? ① Metrics 수집 ② 로그 저장 ③ ETL ④ 컴파일
    **정답:** ①

9.  Grafana의 역할은? ① Dashboard 시각화 ② 이미지 스캔 ③ DB 관리 ④ DNS
    관리 **정답:** ①

10. SLO 99.95%의 Error Budget은? ① 0.05% ② 0.5% ③ 5% ④ 0.005% **정답:**
    ①

11. 무중단에 가장 가까운 DR 구성은? ① Active-Active ② Warm Site ③ Cold
    Site ④ Tape Backup **정답:** ①

12. RPO는 무엇을 의미하는가? ① 허용 가능한 최대 데이터 손실 시점 ② 허용
    가능한 최대 복구 시간 ③ 평균 응답 시간 ④ CPU 사용률 **정답:** ①

13. 장기 운영 VM 비용 절감에 적합한 것은? ① Reserved Instance/Savings
    Plan ② Spot만 사용 ③ VM 증설 ④ Scale Up **정답:** ①

14. 데이터 생성부터 활용까지의 흐름을 추적하는 것은? ① Data Lineage ②
    Metadata ③ RAID ④ Snapshot **정답:** ①

15. Zero Trust의 핵심 원칙은? ① Never Trust, Always Verify ② 내부망은
    신뢰 ③ VPN만 사용 ④ 방화벽 제거 **정답:** ①

16. SPI=0.97의 의미는? ① 일정 지연 ② 일정 단축 ③ 비용 절감 ④ 품질 향상
    **정답:** ①

17. CPI=1.12의 의미는? ① 비용 효율 양호 ② 비용 초과 ③ 일정 지연 ④ 위험
    증가 **정답:** ①

18. API Gateway의 기능이 아닌 것은? ① 컨테이너 오케스트레이션 ② 인증 ③
    라우팅 ④ Rate Limiting **정답:** ①

19. 장애의 근본 원인을 규명하고 재발 방지 대책을 수립하는 활동은? ① RCA
    ② CAB ③ SLA ④ PMO **정답:** ①

20. Enterprise Architecture의 최종 목적은? ① Business-IT Alignment ②
    서버 증설 ③ 회선 증설 ④ 스토리지 증설 **정답:** ①
