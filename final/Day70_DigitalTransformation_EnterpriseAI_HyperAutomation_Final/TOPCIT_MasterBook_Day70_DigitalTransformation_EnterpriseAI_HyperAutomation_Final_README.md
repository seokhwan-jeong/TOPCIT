# 1. Digital Transformation(DX)

DX는 디지털 기술을 활용하여 비즈니스 모델과 업무 프로세스를 혁신하는
활동이다.

핵심 요소 - Cloud - AI - Big Data - IoT - Automation - Platform

성공 요인 - 경영진 지원 - 데이터 기반 의사결정 - 조직 문화 변화

------------------------------------------------------------------------

# 2. Enterprise AI

Enterprise AI는 기업 전반에 AI를 적용하여 업무 효율성과 의사결정을
향상시키는 전략이다.

적용 사례 - AI 챗봇 - 예지보전 - 수요 예측 - 문서 자동 요약 - 코드 생성

------------------------------------------------------------------------

# 3. Hyper Automation

Hyper Automation = AI + RPA + Workflow + Analytics

``` text
User
 ↓
Workflow
 ↓
RPA
 ↓
AI
 ↓
ERP / CRM
```

효과 - 반복 업무 제거 - 비용 절감 - 생산성 향상

------------------------------------------------------------------------

# 4. RPA vs Low-Code / No-Code

  항목   RPA           Low-Code / No-Code
  ------ ------------- --------------------
  목적   업무 자동화   앱 개발
  대상   반복 업무     비즈니스 사용자
  대표   UiPath        Power Apps

# TOPCIT 종합 실전 문제

## 문제1

DX의 핵심 목표는?

① 서버 증설 ② 비즈니스 혁신 ③ RAID 구성 ④ DNS 변경

**정답:** ②

**해설:** DX는 디지털 기술을 활용하여 업무와 비즈니스 모델을 혁신하는
활동이다.

------------------------------------------------------------------------

## 문제2

기업 전반에 AI를 적용하는 전략은?

① Enterprise AI ② ETL ③ RAID ④ VLAN

**정답:** ①

**해설:** Enterprise AI는 조직 전체의 업무와 의사결정에 AI를 적용한다.

------------------------------------------------------------------------

## 문제3

반복 업무 자동화에 가장 적합한 기술은?

① RPA ② SMTP ③ FTP ④ RAID5

**정답:** ①

**해설:** RPA는 규칙 기반 반복 업무를 자동화한다.

------------------------------------------------------------------------

## 문제4 (시나리오)

기업이 AI, RPA, Workflow를 통합하여 업무를 자동화하려 한다. 가장 적합한
개념은?

① Hyper Automation ② CDN ③ VPN ④ SAN

**정답:** ①

**해설:** Hyper Automation은 다양한 자동화 기술을 통합한다.

------------------------------------------------------------------------

## 문제5 (종합)

다음 중 클라우드 네이티브 기술이 아닌 것은?

① Kubernetes ② Docker ③ GitOps ④ RAID Controller

**정답:** ④

**해설:** RAID Controller는 스토리지 기술이며 클라우드 네이티브 기술이
아니다.
