# TOPCIT 실전 문제

## 문제1 (중 \| ★★★★★)

이미지 분류에 가장 적합한 모델은?

① RNN ② CNN ③ Kafka ④ RAID

**정답:** ②

**해설:** CNN은 합성곱 연산을 이용하여 이미지 특징을 효과적으로
추출한다.

------------------------------------------------------------------------

## 문제2 (상 \| ★★★★★)

LLM의 환각(Hallucination)을 줄이는 방법으로 가장 적절한 것은?

① RAID5 ② RAG ③ DHCP ④ Trigger

**정답:** ②

**해설:** RAG는 외부 문서를 검색한 뒤 LLM에 제공하여 정확도를 높인다.

------------------------------------------------------------------------

## 문제3 (상 \| ★★★★★)

Transformer의 핵심 기술은?

① Replication ② Self-Attention ③ RAID ④ MVCC

**정답:** ②

------------------------------------------------------------------------

## 문제4 (시나리오 \| ★★★★★)

기업 내부 문서를 기반으로 답변하는 챗봇을 구축하려 한다. 가장 적합한
구조는?

① CNN ② RAG + Vector DB ③ RAID10 ④ FTP

**정답:** ②

**해설:** RAG는 내부 문서를 검색한 후 LLM이 답변을 생성한다.

------------------------------------------------------------------------

## 문제5 (시나리오 \| ★★★★★)

모델을 재학습하지 않고 응답 품질을 빠르게 개선하려면?

① Fine-tuning ② Prompt Engineering ③ GPU 제거 ④ ETL

**정답:** ②

**해설:** Prompt Engineering은 모델 수정 없이 결과를 개선하는 가장 빠른
방법이다.

------------------------------------------------------------------------
