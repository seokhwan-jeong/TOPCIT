# 1. Cloud Security

클라우드 보안은 데이터, 계정, 네트워크, 워크로드를 보호하는 체계이다.

``` text
User
 │
IAM
 │
Application
 │
Cloud Platform
 │
Infrastructure
```

핵심 원칙

-   최소 권한
-   암호화
-   감사
-   지속적 모니터링

------------------------------------------------------------------------

# 2. Shared Responsibility Model

  영역              CSP   고객
  ----------------- ----- ------
  물리 데이터센터   ✔     
  하드웨어          ✔     
  가상머신 설정           ✔
  애플리케이션            ✔
  데이터                  ✔

IaaS일수록 고객 책임이 많고 SaaS일수록 CSP 책임이 커진다.

------------------------------------------------------------------------

# 3. IAM

IAM은 사용자와 서비스의 접근 권한을 관리한다.

구성 요소

-   User
-   Group
-   Role
-   Policy
-   MFA

실무 원칙

-   Least Privilege
-   MFA 적용
-   정기 권한 검토

------------------------------------------------------------------------

# 4. KMS

KMS(Key Management Service)는 암호화 키를 생성·관리한다.

활용

-   디스크 암호화
-   DB 암호화
-   Object Storage 암호화
-   Secret 관리

대표 서비스

-   AWS KMS
-   Azure Key Vault
-   Google Cloud KMS

------------------------------------------------------------------------

# 5. CSPM / CWPP / CNAPP

  기술    역할
  ------- -------------------------
  CSPM    클라우드 설정 오류 점검
  CWPP    워크로드 보호
  CNAPP   CSPM+CWPP 통합 플랫폼

------------------------------------------------------------------------

# TOPCIT 실전 문제

## 문제1 (중 \| ★★★★★)

Shared Responsibility Model에서 애플리케이션 보안 책임은?

① CSP ② 고객 ③ ISP ④ CDN

**정답: ②**

**해설:** 애플리케이션, 계정, 데이터 보안은 고객의 책임이다.

------------------------------------------------------------------------

## 문제2 (상 \| ★★★★★)

암호화 키를 안전하게 관리하는 서비스는?

① IAM ② KMS ③ CSPM ④ Grafana

**정답: ②**

**해설:** KMS는 키 생성, 저장, 회전(Rotation)을 지원한다.

------------------------------------------------------------------------

## 문제3 (상 \| ★★★★★)

클라우드 설정 오류를 탐지하는 솔루션은?

① CSPM ② CWPP ③ Loki ④ Helm

**정답: ①**

**해설:** CSPM은 잘못된 보안 그룹, 공개 스토리지 등 설정 오류를
분석한다.

------------------------------------------------------------------------

## 문제4 (상 \| ★★★★☆)

Zero Trust의 핵심 원칙은?

① 내부망은 모두 신뢰 ② Never Trust, Always Verify ③ VPN만 사용 ④
방화벽만 설치

**정답: ②**

------------------------------------------------------------------------

## 문제5 (시나리오 \| ★★★★★)

퇴사한 직원 계정이 그대로 남아 있었다. 가장 적절한 개선책은?

① VM 재시작 ② IAM 계정 수명주기 관리와 정기 권한 검토 ③ DNS 변경 ④
스토리지 증설

**정답: ②**

**해설:** IAM 거버넌스와 계정 Lifecycle 관리가 필수이다.
