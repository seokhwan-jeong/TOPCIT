# 1. Big Data란?

빅데이터는 기존 시스템으로 처리하기 어려운 대용량 데이터를 저장·분석하는
기술이다.

## Big Data 5V

  요소       의미
  ---------- ---------------
  Volume     데이터 규모
  Velocity   생성 속도
  Variety    데이터 다양성
  Veracity   데이터 신뢰성
  Value      비즈니스 가치

# 3. HDFS

구성요소

-   NameNode : 메타데이터 관리
-   DataNode : 실제 데이터 저장

특징

-   블록 기반 저장
-   복제(Replication)
-   장애 허용

------------------------------------------------------------------------

# 4. MapReduce

단계

``` text
Input
 ↓
Map
 ↓
Shuffle
 ↓
Reduce
 ↓
Output
```

대용량 배치 처리에 적합하다.

------------------------------------------------------------------------

# 5. Apache Spark

Spark는 메모리 기반 분산 처리 엔진이다.

장점

-   빠른 처리
-   실시간 분석
-   머신러닝 지원
-   Streaming 지원

------------------------------------------------------------------------

# 6. Apache Kafka

Kafka는 분산 이벤트 스트리밍 플랫폼이다.

구성

-   Producer
-   Broker
-   Topic
-   Consumer

활용

-   로그 수집
-   실시간 데이터 처리
-   이벤트 기반 시스템

------------------------------------------------------------------------

# 7. Data Lake vs Data Warehouse

  항목     Data Lake        Data Warehouse
  -------- ---------------- -----------------
  데이터   원본 그대로      정제된 데이터
  구조     Schema-on-Read   Schema-on-Write
  목적     AI/분석          BI/리포트

------------------------------------------------------------------------

# 8. ETL vs ELT

ETL - 추출→변환→적재

ELT - 추출→적재→변환

클라우드 환경에서는 ELT 활용이 증가하고 있다.

# TOPCIT 실전 문제

## 문제1 (중 \| ★★★★★)

HDFS에서 메타데이터를 관리하는 구성요소는?

① DataNode ② NameNode ③ Broker ④ Topic

**정답:** ②

**해설:** NameNode는 파일 위치와 블록 정보를 관리한다.

------------------------------------------------------------------------

## 문제2 (상 \| ★★★★★)

MapReduce보다 메모리 기반으로 빠른 처리가 가능한 기술은?

① FTP ② Spark ③ RAID ④ DHCP

**정답:** ②

**해설:** Spark는 메모리 기반 연산으로 반복 처리와 실시간 분석에
유리하다.

------------------------------------------------------------------------

## 문제3 (상 \| ★★★★★)

실시간 이벤트 스트리밍 플랫폼은?

① Kafka ② Oracle ③ Redis Sentinel ④ Jenkins

**정답:** ①

------------------------------------------------------------------------

## 문제4 (시나리오 \| ★★★★★)

AI 학습을 위해 원본 데이터를 그대로 저장하려 한다. 적합한 저장소는?

① Data Warehouse ② Data Lake ③ RAID5 ④ View

**정답:** ②

**해설:** Data Lake는 다양한 원본 데이터를 그대로 저장한다.

------------------------------------------------------------------------

## 문제5 (시나리오 \| ★★★★★)

매일 경영 리포트를 생성하기 위한 분석 저장소는?

① Data Lake ② Data Warehouse ③ Kafka ④ HDFS

**정답:** ②

**해설:** 정제된 데이터를 기반으로 BI와 리포팅에 적합하다.
